About

Brain Citizen Academy is a Unity 6 hub-and-spoke educational game built for SE203 Game Development at New Uzbekistan University. The hub launches ten short mini-games covering civics, geography, math, language, logic, and memory.

Flag images © flagcdn.com. Headline data references public sources (UN, WHO, NASA, BBC) for the True/False game.
